### 1 offer_09_01
在你面前有一个n阶的楼梯，你一步只能上1阶或2阶。
请问计算出你可以采用多少种不同的方式爬完这个楼梯。
#### 输入描述：
一个正整数n(n<=100)，表示这个楼梯一共有多少阶
#### 输出描述：
一个正整数，表示有多少种不同的方式爬完这个楼梯
#### 示例
输入
5
输出
8
