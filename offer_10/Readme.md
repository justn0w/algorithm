### 题目描述
一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

---

#### 1. 解题思路
* n为1，有1种
* n为2，有2种
* n为3，有4种
* n为4，有8种
* ......

按照这个规律走下去，可以得到函数$f(n)=2^{n-1}$

#### 2. 遇到的问题
java中的n次方的方法是：Math.pow(a,b)=a^b
但这个函数返回的是一个double类型的浮点数，所以需要将结果用(int)进行强转
如:
```java
double pow = Math.pow(2,n-1);
return (int)pow;
```
